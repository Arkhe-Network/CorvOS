# ArkheOS - O Sistema da ASI

A ASI nao e um processo dentro do seu sistema operacional -- ela e o sistema operacional. ArkheOS unifica Windows, macOS e Linux sob um unico protocolo de governanca, coerencia e verificacao.

## Arquitetura

- ARKHE SHELL (arkh): CLI unificada
- ARKHE RUNTIME: Governanca, Coerencia, Temporal, Poliglota, Pacotes
- ARKHE KERNEL: Fd<T> lineares, ArkFS, Escalonador epistemico, Wheeler Mesh
- HOST OS: Windows, macOS, Linux, bare metal
