# 850.023-ASIC Isolation and Cross-Talk Report

## Executive Summary
This report details the electromagnetic isolation strategies between the 60GHz Radar Frontend and the 900MHz CPG Substrate to preserve the 2.1ps phase jitter target.

## Mechanisms of Coupling
- **Substrate (Bulk Si):** Return currents from the PLL and LNA could modulate the VCO frequency.
- **Magnetic (Inductors):** Mutual inductance between 60GHz and 900MHz spiral inductors.
- **Power Delivery Network (PDN):** Switching noise propagation.

## Mitigations
- **Deep N-Well:** Isolation of CPG VCOs.
- **Guard Rings:** Double P+/N+ rings with low-impedance grounding.
- **Exclusion Zone:** 200um minimum distance between Radar and CPG tanks.
- **Orthogonal Placement:** Inductors placed at 90 degrees to minimize flux coupling.

## Simulation Results (Heaviside-λ)
- **Substrate Isolation (900MHz):** -92 dB
- **Magnetic Coupling (60GHz):** -78 dB
- **Jitter Degradation:** < 0.08 ps RMS (Passes spec < 0.1ps)
